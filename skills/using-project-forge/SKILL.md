# Using Project Forge

Project Forge is a persona-driven pipeline that turns vague ideas into implementation-ready architecture through a chain of expert AI prompt engineers.

## Pipeline

1. **forge-intake** (资深产品需求分析师) — classifies the domain, writes a system prompt
2. **creative-director** (资深AI Prompt工程师 + 产品架构师) — probes depth, writes architectural brief
3. **ai-architect** (资深AI Prompt工程师 + 技术架构师) — selects stack, writes ADR
4. **harness-engineer** (资深AI Prompt工程师 + DevOps工程师) — generates scaffold, writes Superpowers handoff

Each step's output is a carefully constructed system prompt for the next step.
