# SvelteKit Harness

Template: `sveltekit`

## Commands

- **install**: `npm install`
- **test**: `npm run test`
- **lint**: `npm run lint`
- **typecheck**: `npm run check`
- **build**: `npm run build`
- **run**: `npm run dev`
- **smoke**: `npm run build`

## Local Setup

1. Clone the repository
2. Run the install command
3. Run verify (test + lint + typecheck)
4. Start local dev with the run command
