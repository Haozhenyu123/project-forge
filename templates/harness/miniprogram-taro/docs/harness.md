# Taro Mini Program Harness

Template: `miniprogram-taro`

## Commands

- **install**: `npm install`
- **test**: `npm run test`
- **lint**: `npm run lint`
- **typecheck**: `npm run typecheck`
- **build**: `npm run build:weapp`
- **run**: `npm run dev:weapp`
- **smoke**: `npm run build:weapp`

## Local Setup

1. Clone the repository
2. Run the install command
3. Run verify (test + lint + typecheck)
4. Start local dev with the run command
