# WeChat Mini Program Native Harness

Template: `miniprogram-native`

## Commands

- **install**: `echo No install step - use WeChat DevTools`
- **test**: `echo No test framework configured`
- **lint**: `echo No lint configured`
- **typecheck**: `echo No type checking configured`
- **build**: `echo Use WeChat DevTools to preview and upload`
- **run**: `echo Open project in WeChat DevTools`
- **smoke**: `echo Verify app.json and project.config.json exist`

## Local Setup

1. Clone the repository
2. Run the install command
3. Run verify (test + lint + typecheck)
4. Start local dev with the run command
