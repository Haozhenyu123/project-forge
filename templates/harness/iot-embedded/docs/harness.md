# IoT / Embedded Harness

Template: `iot-embedded`

## Commands

- **install**: `pio project init`
- **test**: `pio test`
- **lint**: `pio check`
- **typecheck**: `echo Uses compile-time checking`
- **build**: `pio run`
- **run**: `pio run --target upload`
- **smoke**: `pio run`

## Local Setup

1. Clone the repository
2. Run the install command
3. Run verify (test + lint + typecheck)
4. Start local dev with the run command
