# Unity Game Harness

Template: `unity`

## Commands

- **install**: `echo Open project in Unity Hub`
- **test**: `echo Use Unity Test Framework`
- **lint**: `echo Use Rider or Visual Studio analysis`
- **typecheck**: `echo C# compiled by Unity`
- **build**: `echo File > Build Settings in Unity Editor`
- **run**: `echo Press Play in Unity Editor`
- **smoke**: `echo Verify Assets/ and ProjectSettings/ exist`

## Local Setup

1. Clone the repository
2. Run the install command
3. Run verify (test + lint + typecheck)
4. Start local dev with the run command
