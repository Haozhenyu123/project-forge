# Godot Game Harness

Template: `godot`

## Commands

- **install**: `echo Download Godot Engine and open project`
- **test**: `godot --headless --script tests/run_tests.gd`
- **lint**: `echo Use GDScript toolkit or gdformat`
- **typecheck**: `godot --headless --check-only`
- **build**: `godot --headless --export-release Web`
- **run**: `godot --editor`
- **smoke**: `godot --headless --quit`

## Local Setup

1. Clone the repository
2. Run the install command
3. Run verify (test + lint + typecheck)
4. Start local dev with the run command
